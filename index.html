Portfolio

Msoko

Msoko is an e-commerce website for a grocery store to sell their produce. A complete user experience for consumers to easily find and purchase produce On the front end, We designed and styled the app using Bootstrap, HTML, CSS and Ember Paper. On the back end, we used Ember Js, Firebase, and Google Maps API .

DrugStore

DrugStore.A product management system for a pharmacy. It has an Admin page to track store inventory details and and a page where users can purchase the drugs. The Technologies mainly used include Java, Spark, JUnit ,CSS restfulful routing and postgresql for the database.

MyRecipe

MyRecipe is an android application that allows users to search for recipes and save their favorite recipes.Technologies mainly used include: Java, Andriod Studio, Firebase for authentication and Edmam API to fetch the recipes.

Smart Mirror

Smart Mirror is a product to make your life easier, while doing something you already spend time doing each day, like looking at a mirror.It provides the user with weather updates as well as the latest news. The main technologies used are: Pthon for the back end, Tkinter for the Graphical User Interface, Dark Sky API for the weather updates and Google News API for the news update.
