Portfolio

Mawingu Ticketing System

It is a ticketing sytem that uses Kaizala Custom Action Card that will be used by Field Agents to raise Customer Issues from the Field and saves the issues to Dyanamics 365 as new case, assigns the case to a group (Queue) and updates the Field Agent via text message when the Case Status is Updated. Microsoft Kaizala, Microsoft Flow, Dynamics 365 and Azure.

Beauty World

Beauty World is a beuty serviproductce management system for a beauty spa. It has an Admin page to track inventory and customer page where users can purchase the services. The Technologies mainly used include Laravel, Bootstrap, Azure and MySQL for the database.

MyRecipe

MyRecipe is an android application that allows users to search for recipes and save their favorite recipes.Technologies mainly used include: Java, Andriod Studio, Firebase for authentication and Edmam API to fetch the recipes.

Smart Mirror

Smart Mirror is a product to make your life easier, while doing something you already spend time doing each day, like looking at a mirror.It provides the user with weather updates as well as the latest news. The main technologies used are: Pthon for the back end, Tkinter for the Graphical User Interface, Dark Sky API for the weather updates and Google News API for the news update.
